===============================================================================
Movie-Recommendation-System-on-IMDb
===============================================================================

-------------------------------------------------------------------------------
The project done as a part of the EE232E in Spring 2017. 
The goal of the project is split into 3 parts:
 1. Create the graph for the actor/actress network
 2. Create the graph for the movie network
 3. Predict the ratings for movie
For more project details, please read the PDF document (Project 2 Description). 
-------------------------------------------------------------------------------

-------------------------------------------------------------------------------
The .py file can run by Python.
The .R file can run by R.
-------------------------------------------------------------------------------

-------------------------------------------------------------------------------

Directly run problem1Part1.py which is the code for problem 1 Part 1.
Directly run problem1Part2.py which is the code for problem 1 Part 2.
Directly run problem2.py which is the code for problem 2.
